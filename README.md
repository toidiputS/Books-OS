## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (latest LTS recommended)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/toidiputS/Books-OS.git
   cd Books-OS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file (or edit the existing one) and add your AI API keys if applicable:
   ```env
   GEMINI_API_KEY=your_key_here
   ```

4. **Launch the Core:**
   ```bash
   npm run dev
   ```
